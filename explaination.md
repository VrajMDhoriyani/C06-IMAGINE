# Basics of Python Programming Language

This repository demonstrates some fundamental concepts of Python programming including functions, loops, object-oriented programming, and mathematical algorithms.

---

## 1. Creating Functions and Using Loops

The files **`task_normalisation.py`** and **`basic_loops.py`** demonstrate the use of basic Python syntax for defining functions and using loops to iterate over input parameters.

Key concepts covered:

- Defining Python functions
- Using `for` loops to process data
- Performing calculations on datasets

The normalization values are calculated using the general normalization formula on a sample dataset.

Additionally, the program lists all **prime numbers up to a given integer** using the **Sieve of Eratosthenes algorithm**, implemented in the file **`basic_loops.py`**.

---

## 2. Functions and Their Variations

The file **`basic_functions.py`** demonstrates several ways to design and use functions in Python.

Topics covered include:

- Functions with **default arguments**
- Simple mathematical functions
- **Lambda functions (inline functions)**

Lambda functions provide a compact way to create small anonymous functions.

---

## 3. Classes and Methods: Object-Oriented Programming in Python

The file **`basic_class.py`** demonstrates the implementation of **Object-Oriented Programming (OOP)** in Python.

A class named **`Student`** is created with several class methods and attributes.

Key concepts demonstrated:

- Class variables (e.g., `school_name`)
- Object creation
- Class methods
- Special methods (dunder methods)

Important dunder methods used:

- `__init__` → Initializes the object when it is created
- `__str__` → Returns a human-readable representation of the object

These methods allow the class to behave more naturally when objects are created and printed.

---

## Conclusion

This repository provides a simple introduction to fundamental Python programming concepts including:

- Functions
- Loops
- Mathematical algorithms
- Object-Oriented Programming

These examples serve as building blocks for writing more complex Python programs.
